# Operating Model & DevOps

## CI/CD expectations (Lead level)
- Version control for notebooks + SQL
- Environment promotion (dev/test/prod)
- Automated tests: unit + data quality + reconciliation
- Monitoring: pipeline success, data freshness, drift, SLA breaches

## Orchestration
- Fabric: pipelines / orchestration artifacts
- Databricks: Workflows / Jobs
- Snowflake: Tasks + external orchestrators (Airflow)

## Observability
- Define operational dashboards: job latency, failure rates, cost, data freshness
- Standardize alerts for broken SLAs
